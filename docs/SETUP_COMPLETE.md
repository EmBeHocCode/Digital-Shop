## ⚠️ Important Notes

### Security Practices
- ✅ Share code freely with agents (nothing sensitive)
- ⚠️ Verify security suggestions for payments/auth
- ⚠️ Don't share API keys or credentials
- ⚠️ Always test production-critical code

### Quality Assurance
- ✅ Use agents as advisors, not replacements
- ✅ Verify suggestions with your team
- ✅ Test before shipping production code
- ✅ Maintain critical thinking

### Best Practices
- ✅ Provide specific context & goals
- ✅ Iterate with follow-up questions
- ✅ Use multiple agents for complex tasks
- ✅ Share learnings with your team
